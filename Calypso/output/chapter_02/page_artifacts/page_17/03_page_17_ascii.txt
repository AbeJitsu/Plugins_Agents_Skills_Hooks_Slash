│  Property                                                                   │
│                                                                             │
│  [Body Text - 11pt Regular, spans from left margin]                        │
│  In common understanding, property is something that is owned by someone.  │
│  A car is the property of Bill Brown if Bill Brown owns the car. If the    │
│  item is not owned, it is not property. For example, if a car is abandoned │
│  and left to rust in the desert, the car is no longer property, since no   │
│  one claims ownership. Similarly, the planet Jupiter is not property,      │
│  since no one owns it.                                                      │
│                                                                             │
│  [Body Text - 11pt Regular with emphasis]                                  │
│  From a more technical standpoint, property is not only the item that is   │
│  owned but also a set of rights to the item enjoyed by the owner. These    │
│  rights are commonly known as the "bundle of rights."                      │
│                                                                             │
│  ──────────────────────────────────────────────────────────────────────    │

│  [Body Text - 11pt Regular]                                                │
│  In owning property, one has the right to possess and use it as the law    │
│  allows. The owner has the right to transfer ownership of the item (sell,  │
│  rent, donate, assign, or bequeath). The owner may also encumber the item  │
│  by mortgaging it as collateral for debt. Finally, the owner has the right │
│  to exclude others from use of the item.                                    │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  In the example of the car, when Bill Brown bought the car, the car became │
│  his property. Bill now has the right to use the car, sell it to someone   │
│  else, rent it out, give it away, or mortgage it. Bill also has the right  │
│  to prevent others from using his car.                                     │
