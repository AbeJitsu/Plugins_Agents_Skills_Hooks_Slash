│  Periodic tenancy                                                           │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  A periodic tenancy is a leasehold with a fixed period that automatically  │
│  renews unless proper notice is given by either party. The estate may be   │
│  month-to-month, year-to-year, or week-to-week.                            │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  At the end of the six months, the tenant pays, and the landlord accepts   │
│  monthly rent payment. The acceptance of the rent automatically renews the  │
│  leasehold for another six months.                                          │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  The most common form of periodic tenancy is the month-to-month tenancy.   │
│  Monthly tenancies may exist without any written agreement.                 │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  Either party may terminate a periodic tenancy by giving proper notice to  │
│  the other party. Proper notice is defined by state law.                    │
│                                                                             │
│  [H4 Subsection - 11pt Bold]                                                │
│  Estate at will                                                             │
│                                                                             │
│  [Body Text - 11pt Regular with bold terms]                                 │
│  The estate at will, also called a tenancy at will, has no definite        │
│  expiration date and hence no "renewal" cycle. The landlord and tenant     │
│  agree that the will have no specified termination date, provided rent is  │
│  paid and lease conditions are met.                                         │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  For example, a son leases a house to his father and mother "for as long   │
│  as they want to move.                                                      │
│                                                                             │
│  [Body Text - 11pt Regular]                                                │
│  The estate at will is terminated by proper notice, or by the death of     │
│  either party.                                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
